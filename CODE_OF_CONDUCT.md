# Code of Conduct

This project adopts the **Contributor Covenant, version 2.1** as its code of conduct.

The full text is available at:
https://www.contributor-covenant.org/version/2/1/code_of_conduct/

## Scope

This Code of Conduct applies within all community spaces for this project — the GitHub repository (issues, pull requests, discussions), any official project communication channels, and any public spaces where an individual is representing the project.

## Enforcement

Instances of behavior that violate the Code of Conduct may be reported privately to the project maintainers at:

**`tomash.maciag@gmail.com`**

All complaints will be reviewed and investigated promptly and fairly. The maintainers are obligated to respect the privacy and security of the reporter of any incident.
